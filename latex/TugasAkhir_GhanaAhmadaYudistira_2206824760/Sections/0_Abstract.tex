\begin{abstract}
Statute retrieval benchmarks typically evaluate systems at a single query formality level and in a single language. We construct KUHPerdata, the first Indonesian statute retrieval dataset with two query formality levels (case summaries and layperson questions) sharing identical relevance judgments. We evaluate a range of retrieval methods across KUHPerdata and three existing multilingual datasets (BSARD, STARD, IL-PCSR) and find that vocabulary overlap between queries and statutes is the primary predictor of lexical retrieval failure across all four languages. We further observe that co-relevant statutes cluster structurally (median 18 articles apart) while hard negatives are distant (median 297), and that the optimal balance between lexical and semantic signals varies by dataset. Based on these findings, we extend the paragraph-level graph neural network of \citet{ilpcsr} with statute proximity edges and post-training alpha grid search, which we call Prox-GNN. On four datasets, Prox-GNN improves 24--94\% in MRR@10 over the strongest baseline, with consistent performance across both formality levels. Ablation confirms that each modification addresses the failure pattern that motivated it.
\end{abstract}
