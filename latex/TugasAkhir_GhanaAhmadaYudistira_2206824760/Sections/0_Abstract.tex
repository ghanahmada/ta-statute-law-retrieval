\begin{abstract}
Statute retrieval benchmarks evaluate systems at a single query formality level and in a single language, obscuring whether methods are robust across the conditions that real users face. We construct KUHPerdata, the first Indonesian statute retrieval dataset with two query formality levels (case summaries and layperson questions) sharing identical relevance judgments, enabling controlled measurement of formality robustness. Through systematic error analysis across four multilingual datasets (Indonesian, French, Chinese, English), we identify two universal failure patterns: (1) vocabulary overlap between queries and statutes is the sole predictor of lexical retrieval failure, and (2) co-relevant statutes cluster structurally (median 18 articles apart) while hard negatives are distant (median 297). These observations motivate Prox-GNN, which extends paragraph-level graph neural networks \citep{ilpcsr} with statute proximity edges and post-training alpha grid search. On four datasets, Prox-GNN achieves 24--94\% relative improvement over the strongest baseline, with consistent performance across both formality levels. Ablation confirms that each modification addresses the specific failure pattern it targets: proximity edges improve ranking of structurally clustered relevant articles, while adaptive alpha tuning recovers the dataset-specific lexical signal that learned blending destroys.
\end{abstract}
